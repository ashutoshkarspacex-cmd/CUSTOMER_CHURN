This is a customer churn dataset on which EDA has been done and then the Random forest classifier model has been used 
for predicting whether a particular customer has churned or left the services of  some company or not.
LIBRARIES USED:-
SCI-KIT LEARN
NUMPY
MATPLOTLIB
PANDAS
YDATAPROFILING----TO GENERATE AN ONLINE HTML FILE CONTAINING THE EDA OF THE ENTIRE DATASET.
THE RANDOM FOREST CLASSIFIER IS AN ENSEMBLE MODEL WHICH COMBINES SEVERAL DECISION TREES TOGETHER 
AND DECIDES THE OUTPUT BASED ON MAJORITY VOTING.
EACH TREE USES A DIFFERENT SET OF FEATURES AND WORKS ON DIFFERENT PROPORTIONS OF THE TRAINING DATA.
 BAYESIAN OPTIMIZATION CAN BE IMPLEMENTED FOR DECIDING THE DIFFERENT COMBINATIONS OF FEATURES AND 
 THIER VALUES TO GET THE BEST OUTPUT.
 THE RANDOM FOREST CLASSIFIER USES BAGGING WHICH IS A BOOTSTRAP METHOD THAT DIVIDES THE TRAINING DATA 
 AND USES VARIOUS PROPORTIONS WITH DIFFERENT COMBINATIONS TO TRAIN INDIVIDUAL DECISION TREES.
A DECISION TREE MODEL CAN ALSO BE USED BUT THE PROBLEM THERE IS NO BOOTSTRAPPING SO HIGHER CHANCE OF OVERFITTING LINGERS.
